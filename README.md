# mini_project_week5
مشروع هندسة الميزات: بيانات العقارات
مقدمة

يهدف هذا المشروع إلى تطوير مجموعة بيانات العقارات عبر إنشاء ميزات جديدة تساعد على تحسين التحليلات والنماذج التنبؤية.

الميزات المضافة

ميزات المجال (Domain Features):

price_per_sqft – السعر لكل قدم مربع

rooms_per_sqft – عدد الغرف لكل قدم مربع

التفاعلات (Interactions):

price_x_rooms – حاصل ضرب السعر في عدد الغرف

الخصائص متعددة الحدود (Polynomial Features):

تطبيق درجة 2 على price_per_sqft لالتقاط العلاقات غير الخطية

ملاحظات

تم التعامل مع القيم الفارغة أو الصفرية بطريقة آمنة لضمان جودة البيانات.

جميع التحويلات قابلة للتكرار ويمكن استخدامها في إعداد النماذج لاحقًا.
